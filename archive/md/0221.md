 Hamad,
 yes, it is not implemented.
 Use renumf90 as for an animal model.
 Then use another software (i.e., SAS, R, or similar) to substitute the code of each animal by the code of its sire, as available in the pedigree file.
 Then you can use add_animal, if you keep the same pedigree, but remember that the estimated genetic variance will be 1/4 of the total genetic variance. 
 The word add_sire is for pedigrees of the form "sire sire-of-sire maternal-grandsire-of-sire" (e.g. in the blupf90 documentation)

 Andres
